# ภาพรวมเครื่องมือของ Claude Code

Claude Code มอบชุดเครื่องมือในตัวให้กับโมเดลผ่านกลไก tool_use ของ Anthropic API อาร์เรย์ `tools` ของคำร้องขอ MainAgent แต่ละรายการจะมีคำจำกัดความ JSON Schema ที่สมบูรณ์ของเครื่องมือเหล่านี้ และโมเดลจะเรียกใช้ผ่าน content block `tool_use` ในการตอบกลับ

ต่อไปนี้คือดัชนีจำแนกตามหมวดหมู่ของเครื่องมือทั้งหมด

| เครื่องมือ | วัตถุประสงค์ |

| [TaskOutput](Tool-TaskOutput.md) | รับผลลัพธ์ของงานเบื้องหลัง |

| [TaskUpdate](Tool-TaskUpdate.md) | อัปเดตสถานะงาน ความสัมพันธ์การพึ่งพา ฯลฯ |

## ความสำคัญใน cc-viewer

การเรียกใช้เครื่องมือเป็นหนึ่งในข้อมูลที่สังเกตได้สำคัญที่สุดในบันทึกคำร้องขอของ Claude Code อาร์เรย์ `tools` ของคำร้องขอ MainAgent แต่ละรายการกำหนดชุดเครื่องมือที่โมเดลใช้ได้ ในขณะที่ content block `tool_use` / `tool_result` ในการตอบกลับบันทึกการเรียกใช้เครื่องมือจริงและค่าที่ส่งคืน cc-viewer สร้างสายการเรียกใช้เครื่องมือที่สมบูรณ์ใน Chat Mode โดยการวิเคราะห์ข้อมูลเหล่านี้

การเปลี่ยนแปลงคำจำกัดความเครื่องมือ (เช่น การเพิ่ม/ลบเครื่องมือจากการเชื่อมต่อ/ตัดการเชื่อมต่อ MCP server) จะทำให้เกิดการสร้างแคชใหม่ ดูรายละเอียดที่ [CacheRebuild](CacheRebuild.md)
