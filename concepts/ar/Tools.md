# نظرة عامة على أدوات Claude Code

يوفر Claude Code مجموعة من الأدوات المدمجة للنموذج عبر آلية tool_use في Anthropic API. تحتوي مصفوفة `tools` في كل طلب MainAgent على تعريفات JSON Schema الكاملة لهذه الأدوات، ويستدعيها النموذج في الاستجابة عبر كتل محتوى `tool_use`.

فيما يلي فهرس مصنف لجميع الأدوات.

| الأداة | الغرض |
|--------|--------|
| [Task](Tool-Task.md) | بدء وكيل فرعي (SubAgent) لمعالجة المهام المعقدة متعددة الخطوات |
| [TaskOutput](Tool-TaskOutput.md) | الحصول على مخرجات المهام الخلفية |

## الأهمية في cc-viewer

استدعاءات الأدوات هي من أهم البيانات القابلة للرصد في سجلات طلبات Claude Code. تحدد مصفوفة `tools` في كل طلب MainAgent مجموعة الأدوات المتاحة للنموذج، بينما تسجل كتل محتوى `tool_use` / `tool_result` في الاستجابة استدعاءات الأدوات الفعلية ونتائجها. يقوم cc-viewer بتحليل هذه البيانات لإعادة بناء سلسلة استدعاءات الأدوات الكاملة في Chat Mode.

التغييرات في تعريفات الأدوات (مثل إضافة/إزالة أدوات بسبب اتصال/انقطاع خادم MCP) تؤدي إلى إعادة بناء الذاكرة المؤقتة، راجع [CacheRebuild](CacheRebuild.md).
